PCL

A key component of building cross-platform applications is being able to share code across various platform-specific projects. However, this is complicated by the fact that different platforms often use a different sub-set of the .NET Base Class Library (BCL), and therefore are actually built to a different .NET Core Library Profile. This means that each platform can only use class libraries that are targeted to the same profile, so they would appear to require separate class library projects for each platform.
